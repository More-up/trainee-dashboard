# 技能実習生エンゲージメントアンケートシステム

技能実習生の職場環境・生活状況を把握するための多言語対応アンケートシステムです。

## 📊 システム概要

### アンケート内容
- **質問数**: 35問
- **カテゴリー**: 8カテゴリー
  - 業務・職場環境
  - 給与・待遇
  - 家族・プライベート事情
  - 人間関係
  - 日本語・コミュニケーション
  - 文化・価値観
  - 生活環境
  - キャリア・将来の見通し

### 機能
- 従業員コード管理（1〜20）
- 国籍別自動言語切り替え
- 月別重複回答チェック
- カテゴリー別スコア集計
- AI国籍別文化分析
- CSV出力・印刷機能

## 📁 主要ファイル

- `index.html` - アンケート画面
- `app.js` - アンケートロジック
- `translations.js` - 多言語翻訳データ
- `style.css` - スタイルシート
- `admin-login.html` - 管理者ログイン
- `admin-dashboard.html` - 管理者ダッシュボード
- `admin.js` - ダッシュボードロジック

## 🚀 使い方

### 技能実習生向け
1. アンケートページにアクセス
2. 社員番号と国籍を選択
3. 35問の質問に回答
4. 送信ボタンをクリック

### 管理者向け
1. 管理者ログインページにアクセス
2. 認証情報を入力してログイン
3. ダッシュボードでデータを確認・分析
4. 必要に応じてCSVエクスポート
